# RMMV_CRTAPlugins

### 概要 - Overview -
RPGツクールMV(RPG Maker MV)で使用可能なプラグインです。  
各プラグインの利用方法はソースのヘッダコメントを参照してください。  
また、以下のサイトにも利用方法を掲載しています。

<http://studio.cretia.net/blog/664>

### 提供プラグイン - Plugins -

* CRTA_ScreenPicture  
ピクチャを使用したスクリーンコマンドプラグイン

* CRTA_TimerManager  
タイマー制御プラグイン
